## Introducción 

En esta unidad vas a aprender sobre estructuras de datos. Las estructuras de datos son formas de 
organizar y almacenar datos en una computadora para que puedan ser utilizados de manera eficiente. 
Inicialmente, vamos a analizar juntos cómo se implementa la estructura de datos llamada lista enlazada. 
Luego analizaremos cómo aplicarla a un problema concreto relacionado con la programación creativa. 
Finalmente, te voy a proponer que transfieras lo aprendido a una nueva estructura de datos llamada cola. 
Implementarás la funcionalidad de la cola y aplicarás esta estructura de datos a la solución de un 
problema de programación creativa.

El lenguaje que estamos usando para explorar los conceptos en este curso es C++. Este lenguaje cuanta 
en su biblioteca estándar con la implementación de listas enlazadas y colas. Sin embargo, en este curso 
vamos a implementar estas estructuras de datos desde cero para que puedas entender cómo funcionan y 
cómo se pueden aplicar a problemas concretos. Adicionalmente, vas a conocer un framework de programación 
creativa llamado openFrameworks que te permitirá aplicar los conceptos aprendidos en la solución de
problemas creativos.


## ¿Qué aprenderás en esta unidad?

En esta unidad aprenderás a:

- Implementar una lista enlazada en C++.
- Aplicar una lista enlazada a un problema de programación creativa.
- Implementar una cola en C++.
- Aplicar una cola a un problema de programación creativa.
